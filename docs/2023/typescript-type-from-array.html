TypeScriptで配列から配列の各値のどれか１つに該当する型を生成する方法

下記のようにします。

const list = ['foo', 'bar', 'baz'] as const
type Values = typeof list[number]
// type Values = "foo" | "bar" | "baz"

各値の特定のプロパティに該当する型を生成する場合は下記のようにします。

const list = [
  { value: 'foo'},
  { value: 'bar'},
  { value: 'baz'},
] as const
type Values = typeof list[number]['value']
// type Values = "foo" | "bar" | "baz"
